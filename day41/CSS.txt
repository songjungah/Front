[CSS]
- C(Cascading) S(Style) S(Sheets)
- Cascading : 위에서 아래로 쏟아진다는 뜻
- Style : 형태, 꼴, 모양 뜻
- Sheets : 씌우는 천/막/면

1. 위에서 아래로 모양을 꾸미기 시트를 덮는다.
2. 주로 HTML 언어를 꾸미는 목적으로 사용된다.
3. CSS는 (HTML)종속형 스타일 언어, 즉 HTML 안에서 같이 사용되는 언어
4. HTML에서 CSS 적용하는 방법
    
    - 방법1 : <마크업 Style = "값">, 선택자가 없다
        - 적용할 마크업에 Style 속성을 이용하여 css 속성값을 대입한다.
        - <마크업 style="속성:값;">
    
    - 방법2 : <head> 마크업에 <style> 마크업을 사용하는 방법
        - <head>
            <style>
                선택자 {속성명 : 속성값;}
            </style>
          </head>

    - 방법3 : <head> 마크업에 <link> 마크업을 이용한 css 파일 불러오기 ★권장
        - <head>
            <link rel="stylesheet" href="css파일경로(확장자 포함)" />
          </head>


[CSS 파일 만들기]
1. 생성할 CSS 파일의 위치폴더를 오른쪽 클릭 -> [new]
2. css 파일명을 입력 후 생성한다.(확장자 포함)

[텍스트 관련 속성]
1. font-family : "글꼴"
2. font-size : 글꼴크기 px;
3. font-style : italic;             : 글꼴 기울기
4. font-weight : bold;              : 글꼴 두께
5. color : (영문)색상명;             : 글꼴 색상
6. word-spacing : 크기px;           : 단어간격, 어간
7. letter-spacing : 크기px;         : 글자간격, 자간
8. line-height : 배율%              : 줄간격, 기본값:150%
9. text-align : center/left/right   : 구역내 내용물 가로 정렬
10. text-decoration : 줄종류;        : 내용물의 줄긋기 
    - underline : 밑줄 / overline : 윗줄 / line-through : 취소선
11. text-shadow : x축 y축 흐림정도 색상 : 글쫄 그림자

[박스 관련 속성]
1. border : 선굵기px 선종류 선색상      : 테두리 적용
    - 선종류 : solid 실선, dotted 실선, dashed 파선, double 이중선
2. border-radius : 크기px;             : 둥근모서리
3. box-shadow : x축 y축 흐림정도 색상   : 박스 그림자
4. width : 크기px;                     : 구역의 가로 사이즈
5. height : 크기px;                    : 구역의 세로 사이즈
6. margin : 크기px;                    : 바깥여백, 테투리와 근접한 구역 사이 여백
    - margin 0 auto : ☆구역 가로 가운데 정렬
7. padding : 크기px;                   : 안쪽여백, 테두리와 내용물 사이 여백
8. background-color : 색상;            : 구역 배경색

[display 속성]
: 마크업들을 배치하는 방법을 정의하는 속성
1. 종류
    - block         : 지정한 마크업이 한줄을 차지하는 배치 방법
        - 대표 마크업 : div, h, p, ul, li 등
        - width, height 적용 가능 / margin, padding 적용 가능
    - inline        : 지정한 마크업이 한줄에 포함되는 배치 방법
        - 대표 마크업 : span, a 등
        - width, height 불가능 / margin, padding 좌우만 적용 가능
    - inline-block  : 지정한 마크업이 한줄에 포함되는 배치 방법
        - 대표 마크업 : input, button 등
        - width, height 적용 가능 / margin, padding 적용 가능
    - none          : 지정한 마크업을 표시 안함/숨기기
